Visualization of the parking citation tickets in the Los Angeles Metro Area.

Front End contributions: implementation of new header and search components design, map zoom in/out based on selected area,
search boundary bug fix, and leading the efforts for the code restructure and new design implementation.

Back End contributions: code restructure, dockerization of Node and PostgreSQL for development and data science,
automated CD process with GitHub Actions, addition of backend unit testing (to be added to an automated CI process),
and creation of a staging environment on an AWS EC2 instance.
